欢迎来到我的 **《欧陆风云4》** 个人攻略网站项目！  
这是一个静态网页，记录了我在游戏中的战报、实用代码、君主特质等信息，方便自己回顾，也作为wiki倒下时的备份，欢迎各位使用！


## 📁 项目结构
```plaintext
├── ardabil.html # 阿尔达比勒战报页面
├── consoleCode.html # 控制台代码集合
├── usefulEvent.html # 实用控制台事件
├── goodTrait.html # 正面君主特质
├── badTrait.html # 负面君主特质
├── picture.html # 原版加载图片资源下载页
├── index.html # 网站首页
├── pjCSS/ # 自定义样式
├── pjJS/ # 自定义脚本
├── img/ # 图片资源（旗帜、图标、战报截图等）
└── bootstrap-* / bootstrap-icons-* # Bootstrap 相关依赖
```

## ✨ 主要功能

- 📜 **国家战报**：目前仅有阿尔达比勒——沙汗沙一篇战报，随着时间不断更新。
- 🛠️ **工具页面**：
    - 控制台代码速查
    - 实用事件 ID 列表
    - 正面 / 负面君主特质说明
    - 图片资源下载（本地备份用,也可用于网页开发美术资源）
- 🔍 **关键词搜索**：支持在表格,首页中快速筛选成就tag，控制台与事件代码。
- 📱 **响应式布局**：基于 Bootstrap 5，在手机、平板和电脑上均可使用(不推荐在手机上观看战报页面)。

## 🚀 如何本地运行

1. 在release页面下载网页离线压缩包
   [Releases 页面](https://github.com/qsxes/eu4Blog_ncnmb/releases/tag/webBackUp)
2. 解压打开index.html即可(本网站完全静态，无需后端服务)
